# Practical-ML-assignment
Week 4 peer review assignment submission for practical machine learning course (course 8 of of Johns Hopkins Data Science Specialisation) 

The goal of this analysis is to use the Weight Lifting Exercises dataset to train a predictor that can determine how well a dumbell bicep curl is performed, and help identify common errors in weight lifting technique.

The data used to train the model is the Weight Lifting Exercises dataset.
